## 🚀 Dự án tiêu biểu (Featured Projects)

### 🛡️ Hệ thống DevSecOps & Web Application Firewall (WAF)
* **Mô tả:** Xây dựng hệ thống bảo vệ ứng dụng web với hạt nhân là **ModSecurity (WAF)** kết hợp giám sát bảo mật.
* **Kiến trúc:** Triển khai trên AWS EC2, đóng gói bằng Docker và sử dụng Jenkins cho CI/CD pipeline. Toàn bộ hệ thống kết nối Zero Trust thông qua Tailscale Mesh VPN.

### 🛒 VulnShop - E-commerce Pentest Lab
* **Mô tả:** Một trang web thương mại điện tử cố tình chứa các lỗ hổng bảo mật được tôi tự tay phát triển (Custom-built).
* **Mục tiêu:** Tạo môi trường lab thực tế để rèn luyện kỹ năng khai thác các lỗ hổng như **OS Command Injection**, SQL Injection và Unrestricted File Upload.

### 🥋 Hệ thống quản lý cửa hàng thiết bị Karate
* **Mô tả:** Phần mềm bán hàng tích hợp dashboard quản trị bằng Python & MySQL, áp dụng các thư viện phân tích dữ liệu để làm báo cáo nhân khẩu học và số liệu kinh doanh.

### 🎮 Unity 2D Asteroid Game
* **Mô tả:** Dự án game 2D phát triển trên Unity, tập trung vào việc xử lý tương tác vật lý, hệ thống spawn thiên thạch và quản lý scene UI với singleton pattern.
